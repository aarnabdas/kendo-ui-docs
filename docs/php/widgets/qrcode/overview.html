QRCode

The QRCode for PHP is a server-side wrapper for the Kendo UI QRCode widget.

Getting Started

Here is how to configure a simple Kendo UI QRCode:

  1. Follow the steps from the introduction - include the autoloader, JavaScript and CSS files.

  2. Add a qrcode:

    $qrcode = new \Kendo\Dataviz\UI\QRCode('qrcode');
$qrcode->value("foo");

  3. Choose the appropriate error correction level and encoding.

    $qrcode = new \Kendo\Dataviz\UI\QRCode('qrcode');
$qrcode->value("foo")
       ->errorCorrection("H")
       ->encoding("UTF_8");

  4. Output the chart by echo-ing the result of the render method.

    echo $qrcode->render();

Accessing an Existing QRCode

You can reference an existing QRCode instance via jQuery.data(). Once a reference has been established, you can use the API to control its behavior.

Accessing an existing QRCode instance

    //Put this after your Kendo QRCode for PHP render() call
    <script>
    $(function() {
        // Notice that the Name() of the qrcode is used to get its client-side instance
        var qrcode = $("#qrcode").data("kendoQRCode");
    });
    </script>
