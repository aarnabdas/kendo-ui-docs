Barcode

The Barcode for PHP is a server-side wrapper for the Kendo UI Barcode widget.

Getting Started

Here is how to configure a simple Kendo UI Barcode:

  1. Follow the steps from the introduction - include the autoloader, JavaScript and CSS files.

  2. Add a barcode with the default type (Code39):

    $barcode = new \Kendo\Dataviz\UI\Barcode('barcode');
$barcode->value("foo");

  3. Change the symbology to match your scenario:

    $barcode = new \Kendo\Dataviz\UI\Barcode('barcode');
$barcode->value("foo");
$barcode->type("ean8");

  4. Output the chart by echo-ing the result of the render method.

    echo $barcode->render();

Accessing an Existing Barcode

You can reference an existing Barcode instance via jQuery.data(). Once a reference has been established, you can use the API to control its behavior.

Accessing an existing Barcode instance

    // Put this after your Kendo Barcode for PHP render() call
    <script>
    $(function() {
        // Notice that the name of the barcode is used to get its client-side instance
        var barcode = $("#barcode").data("kendoBarcode");
        barcode.value("foo");
    });
    </script>
